A library of icons and shapes for drawing beautiful [Cell-Based Architecture (CBA)](https://github.com/wso2/reference-architecture/blob/master/reference-architecture-cell-based.md) diagrams on [Excalidraw](https://excalidraw.com)

## How to use it

**Option 1.** Click on the following [installation link](https://excalidraw.com/?addLibrary=https%3A%2F%2Fraw.githubusercontent.com%2Fexcalidraw%2Fexcalidraw-libraries%2Fbinura-g-cell-based-architecture-1763096739352%2Flibraries%2Fbinura-g%2Fcell-based-architecture.excalidrawlib%3Fraw%3Dtrue) to automatically add these shapes and icons to your Excalidraw Personal Library.

**Option 2.** Download the `cba.excalidrawlib` file in this repositosity, open [excalidraw](https://excalidraw.com) (or [excalidraw+](https://app.excalidraw.com)) and open the file via Library > Menu > Open on the right-hand side navbar (see the screenshot below).
